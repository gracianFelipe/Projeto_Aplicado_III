# Pasta `notebooks/`

Esta pasta reúne o notebook principal utilizado no desenvolvimento do projeto.

## Notebook principal

- **Arquivo:** `PA3_Notebook_Final_Recomendacao_Restaurantes.ipynb`

## Conteúdo do notebook

O notebook documenta as principais etapas técnicas do projeto, incluindo:

- leitura e compreensão da base de dados;
- análise exploratória;
- limpeza e preparação dos dados;
- construção da representação textual dos restaurantes;
- vetorização com TF-IDF;
- cálculo de similaridade entre restaurantes;
- implementação do modelo inicial;
- construção do baseline;
- refinamento do pipeline;
- comparação entre abordagens;
- geração das figuras e tabelas utilizadas no relatório final.

## Observação

O notebook foi desenvolvido no Google Colab. Para sua reprodução, pode ser necessário ajustar o caminho do arquivo CSV conforme o ambiente utilizado.
